what is HTML?

HTML(hyper text markup language) is the standard markup language for creating web pages. HTML elements tell browser how to display the content.

What is a basic structure of an HTML document?

  • !DOCTYPE html tag ;this tag is used to tells the html version.
  • html  this is the root element of an HTML page
  • head the head element contains meta information about html page.
  • title the title element specifies a title for html page.(which is shown in the pages tab.
  • body  the body element is a container for all visible contents,such as headings,paragraphs,images,hyperlinks,etc.
  • h1   h1 element difines a large heading
  • p  the p element difines a paragraph.
